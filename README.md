# 🎵 MoodMelody – Music Mood Tracker  

> Track your mood and discover music that matches your emotions.

---

## 📌 Description  

MoodMelody is an Android application that helps users track their daily moods and receive music recommendations based on how they feel.

The app solves a simple problem: users often don’t know what music suits their current emotional state. By combining mood tracking with music suggestions, it creates a more personalized and engaging experience.

---

## 🚀 Features  

- 🎭 Select and log your current mood  
- 🎧 Get music recommendations based on mood  
- 📱 Clean and intuitive Android UI  
- 🔄 Smooth user interaction  
- 📊 Mood tracking (if implemented)  
- 💡 Simple and user-friendly experience

## 📥 Installation  

### 1. Clone the Repository  

```bash
git clone https://github.com/D-sasmita/moodmelody-music-mood-tracker.git
cd moodmelody-music-mood-tracker
```

### 2. Open in Android Studio  

- Open Android Studio  
- Click **Open Project**  
- Select the cloned folder  

### 3. Sync Gradle  

- Let Gradle sync automatically  
- Or click:  
  **File → Sync Project with Gradle Files**

### 4. Run the App  

- Connect a device or start an emulator  
- Click ▶️ Run
